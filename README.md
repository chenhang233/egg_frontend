# react完成的个人项目

创建项目引导 [Create React App](https://github.com/facebook/create-react-app).

## 启动项目:

运行:

### `npm start` 或者 `yarn start`



### `npm run build `或者 `yarn build`

使用脚手架提供的基本webpack配置打包前端文件

### `npm run eject`

不需要,自定义配置
